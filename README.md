---

## Описание

Веб-калькулятор для сравнения налоговой нагрузки по режимам УСН и АУСН для селлеров маркетплейсов. Определяет наиболее выгодную систему налогообложения на основе входных данных клиента.

## Входные данные

| Параметр | Описание |
|---|---|
| Доходы | Годовая выручка, ₽ |
| Расходы | Годовые расходы, ₽ |
| ФОТ | Фонд оплаты труда, ₽ |
| Ставка травматизма | % (обычно 0.2%) |
| Форма организации | ИП или ООО |

## Сравниваемые режимы

- **АУСН (Доходы)** — 8%, без страховых взносов
- **АУСН (Доходы − Расходы)** — 20%, мин. налог 3%
- **УСН (Доходы)** — 6%, НДС 5% при доходах > 60 млн
- **УСН (Доходы − Расходы)** — 15%, мин. налог 1%

## Ставки 2025 года

| Константа | Значение |
|---|---|
| Фиксированные взносы ИП | 57 390 ₽ |
| Взносы 1% с превышения 300 тыс. | 1% × (доход − 300 000) |
| Взносы за работников | 30% от ФОТ |
| Фиксированный травматизм АУСН | 2 959 ₽/год |
| НДС (УСН при доходах > 60 млн) | 5% включено в цену |

## Запуск

Открыть `index.html` в браузере — готово. Не требует сервера или сборки.

## Встраивание в WordPress

**Вариант 1 — Custom HTML блок:**  
Скопировать содержимое `<style>`, разметки и `<script>` тегов в блок «Произвольный HTML» в Gutenberg.

**Вариант 2 — iframe:**
```html
<iframe src="path/to/index.html" width="100%" height="800" frameborder="0"></iframe>
```

**Вариант 3 — отдельная страница:**  
Загрузить `index.html` на сервер и дать клиентам прямую ссылку.
